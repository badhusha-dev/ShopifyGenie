You are an expert Shopify App Developer.  
I want you to create a simplified Shopify App using the following tech stack:

Frontend: React + Bootstrap for styling  
Backend: Express.js (Node.js)  
Storage: In-memory arrays/objects or JSON file persistence for demo  
Shopify Integration: Shopify Admin API (REST + GraphQL) + Webhooks  
Auth: Shopify OAuth 2.0  

## 🎯 App Features (MVP)

1. Inventory Management
   - Sync products from Shopify via API.
   - Track available quantity in memory/JSON.
   - Show low-stock alerts if quantity < 10.

2. Loyalty Points System
   - Assign points on orders ($1 spent = 1 point).
   - Store points per customer in memory/JSON.
   - Display points balance in customer dashboard.

3. Basic Subscriptions
   - Allow customers to subscribe to a product.
   - Save subscription status + renewal date in JSON.
   - Show subscriptions in customer dashboard.

4. Shopify API Integration
   - Import product catalog.
   - Retrieve customer orders.
   - Webhook for new orders → update inventory & loyalty points.

5. Customer Dashboard
   - Display past orders.
   - Show loyalty points balance.
   - Show subscription status.

6. Reports
   - Display total sales.
   - Show top-selling product.
   - Show total loyalty points issued.
